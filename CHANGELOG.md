# Changelog

## [0.1.0] - 2026-05-03

### Initial GitHub release

- Windows-first live call translator with VB-CABLE A+B routing.
- Deepgram Nova-3 streaming STT.
- Groq `llama-3.3-70b-versatile` translation.
- AI Assistant with ChatGPT / Codex, OpenRouter, Groq, and auto fallback providers.
- Quick and detailed answer suggestions from the live transcript.
- Optional web context for detailed assistant answers.
- Piper local TTS voices with in-app voice downloads.
- Microsoft Edge Neural voice support through `edge-tts`.
- Browser Monitor and Windows system loopback capture helpers.
- Flask UI for settings, transcript, bookmarks, export, history, and summaries.
- Elixir supervision around the Rust audio engine.
- Windows setup and run scripts: `setup_windows.ps1` and `run_windows.ps1`.
